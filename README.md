# Landing Page Club Deportivo Jaguar

Uso de IA en el desarrollo de la page
Este proyecto fue desarrollado con asistencia de Claude (Anthropic) como herramienta de IA para:

Generación y revisión de código HTML y CSS
Configuración de EmailJS y Google Apps Script
Aplicación de buenas prácticas semánticas y de accesibilidad
Resolución de errores y debugging

## Tecnologías
- HTML5
- CSS3

## Descripción
Landing page de contacto del Club Deportivo Jaguar.

## Accesibilidad y buenas prácticas

Contraste de colores siguiendo criterios WCAG AA
Etiquetas alt en todas las imágenes
Estructura semántica HTML5
Tipografía legible (Bebas Neue + Barlow)
Variables CSS para consistencia visual
